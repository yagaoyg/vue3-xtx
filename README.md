## 项目部署方法
  本项目未经打包 请先将仓库克隆到本地

  项目克隆至本地后 使用 vscode（或者其他ide、编辑器）打开项目

  请确保你的电脑中安装有 node.js 并且为最新版本

  打开项目后 在终端中先输入以下命令
  
  `npm i`

  等待 npm 包管理器下载所有的包 包下载完成后 在终端中输入以下命令启动项目

  `npm run dev`
  
  启动后根据终端提示或在浏览器中输入 `localhost:5173` 即可进入项目
